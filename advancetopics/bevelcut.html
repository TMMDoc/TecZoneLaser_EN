Bevel Cut

To prepare a sheet metal edge for welding, fabricators often design bevel cuts on the sheet metal. Beveled edges increase the weld surface area, which facilitates material penetration on thick parts and makes welds stronger and more resistant to stress.

Bevel cut is an optional feature for TruLaser machines that tilt the cutting unit for deep bevel cuts on parts. The required NC code will be generated by TecZone Laser.

Figure 1. Bevel Cut Part

Generating Bevel Cut

Chamfer option is enabled for sheet thickness greater than 3 mm.

When a laser machine with the bevel cut option is selected for tooling, the chamfer section becomes visible in the Laser Cut panel.

The top and bottom chamfer angles are allowed in the range of 0 to 45°.

Chamfer angle cannot be negative.

A top chamfer or bottom chamfer is considered active only when both the angle and depth for the respective chamfers are greater than zero.

Multiple cuts might be required for a bevel. The order of cuts is as follows:

  1. Bottom chamfer cut.

  2. Vertical cut.

  3. Top chamfer cut.

TecZone Laser supports four different types of bevel cuts:

Figure 2. Bevel Types

For a V or DV bevel type it is not necessary to make a vertical cut at the tip of the bevel (See cut number 2). In case, a vertical cut is required the user needs to select the option Force vertical cut.

The path of the tool center point on the top surface for making these cuts is shown below. These paths are calculated taking the chamfer angle, depth, and beam width into consideration. Below, you can see all the three cuts of a square hole, along with their chamfer settings.

Figure 3. Tool path

The vertical cut is shown in standard laser cutting colour. The chamfer cuts which are made with tilted heads are shown with a yellow highlight around them.

Corner Treatment on Bevel Cut

The corner treatment describes the geometry if two chamfer on different faces are connected. The default corner treatment for bevel cutting is Conical Rounding. The user can modify the corner treatment of a bevel cut similar to normal cuts.

Select a point close to the corner and choose the required corner treatment .

Figure 4. Corner treatment types

Bevel Cut for Split Contours

It is possible to define the required chamfer angle at various sections of the contour. When a sheet metal part is tooled with bevel cut, click on the bevel contour again. The Change chamfer…​ option is visible under Advanced section of the Laser Cut panel.

This option will appear only if the following conditions are met:

  • Selected contour should be a closed one.

  • The selected laser cut should have a default chamfer angle defined in Chamfer section of Laser Cut panel.

  • The selected laser cut should have only one chamfer angle (i.e) either top chamfer or bottom chamfer.

  • Based on the thickness, the depth changes.

When this Change Chamfer…​ is selected, the following properties appears under Advanced section:

The following example explains chamfer settings:

  • Create a rectangle with outer contour 100x100 mm and inner contour 50x50 mm.

  • Edit sheet size to 10 mm and select a bevel cut laser machine.

  • Go to Cut setting → Cut CAM → Laser CAM and change the process for open polylines to Cut.

  • Process the Laser tooling and select the inner contour to access the Laser Cut panel.

Chamfer: Apply the Top chamfer angle to 30° and Top chamfer depth to 5 mm.

The chamfer angle applies to the entire contour, in counter-clockwise direction.

To change the chamfer angle for a particular section of the contour, click the inner contour again and select change chamfer…​ option.

Apply the new chamfer angle and select a point in the contour. By default, the new chamfer is applied from the selected point to the rest of the contour. Clicking again on the point will remove the new chamfer anf

The new angle will be applied to the rest of the contour.

Continuous transition: If this option is selected, the laser head angle will be interpolated along the segment and the head angle does not change abruptly at the change point.

Continuous transition is not applicable for arc or circle segments and at the corners.

Figure 6. With continuous transition

Corner offset: This option is used to change the corner offset distance at which the chamfer angle changes. When set to 0, the chamfer angle change will start from the corner.

Further, the direction in which the changed chamfer angle is applied is the direction of the contour in drawing and not the direction of cutting. The user can remove an existing angle change point by clicking on the change point. The cursor changes to a cross mark to remove the existing points.

Microjoints in Bevel Cut

It is possible to add microjoints by selecting the respective contour.

Click the contour on a part and navigate to Contour panel.

If the bevel tooling of the contour contains straight cut, then the user can click on the contour to insert a microjoint to the straight cut of the bevel. If a microjoint joint already exists, it will be removed.

Add, Reduce, Remove All options will be disabled if the contour has bevel cuts. Microjoints cannot not be added for V-bevel cuts.
