DetectLine

DetectLine accurately detects work piece position using measurements acquired from the inner contours or outer sheet edges (measure points).

TecZone Laser uses measuring program that contains the technology table TC_Sheet_Measure to detect measure points.

An accuracy of up to ±0.1 mm is achieved. At least two measure points are required for DetectLine to work.

To enable DetectLine feature, go to machine database and select the DetectLine checkbox under Settings.

To change the default sheet handling characteristics, navigate to Settings → Machine Defaults → Sheet handling.

Procedure

  • Load sheet.

  • Make sure DetectLine option is enabled in machine database.

  • Go through 'workflow' in TecZone Laser to complete 'post process' to generate NC code.

  • Transfer the NC code to machine.

  • Run the program on machine.

When DetectLine option is selected, the NC code generated includes the Sheet_Measure program to capture the coordinates of the loaded sheet.

When the program is run, Sheet_Measure program populates the sheets coordinates into TC_Sheet_measure.

DetectLine for Cut to Size Parts

For a part where blank is already cut to size, the following pre-cut geometries are eligible for measure points:

  1. Corners of the sheet

  2. Center of pre-cut circular holes

  3. Right angle corner of any pre-cut polygonal holes

Default Selection of Measure Points

Only circular holes with diameter less than 100 mm are selected as default measure points.
Only two measure points are allowed per layout. If a third measure point is selected, the first one will be removed.
Two outside corners of a layout can not selected as measure points.

In the example layout below, we have three pre-cut geometries. A circle with diameter 50 mm, another circle with diameter 216 mm and a square hole with 100 x 100 mm.

Regardless of these, the bottom left corner of the layout and pre-cut circular hole with diameter 50 mm is selected as default measure points.

Change Measure Points

To change the measure points, click on the edge of the layout to bring the layout panel and select change measure points…​ option.

  • Hover over the change measure points…​ option to see the geometry and placement of the existing measure points.

  • To change measure points, click on any of the eligible measure points.

  • To remove measure points, click on the existing measure point.

  • To copy measure points to a different layout on the same sheet, click copy measure points option.
