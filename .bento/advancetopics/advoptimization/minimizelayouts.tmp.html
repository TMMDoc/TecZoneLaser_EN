It is desirable to have a smaller number of different layouts computed by the nesting engine. That is, create layouts whose repeat count is high. This is like a secondary goal for the nesting engine. TecZone Laser has the option to direct the nesting engine to minimize the number of different layouts, possibly resulting in a slight decrease in nesting efficiency.
