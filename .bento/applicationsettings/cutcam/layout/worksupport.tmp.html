Distance between slats - This setting is to state the distance between slats on the machine.

Offset of the first slat from machine origin - This setting is used to define the offset of the first slat from the machine origin.

Offset of the first pin in a slat - This setting is used to define the offset of the first pin in a slat.

Distance between support pins in a slat - This setting is used to define the distance between the support pins in a slat.

Slats are along X-axis (horizontal) - Enable this if the slats are horizontal along the X-axis.

Pins in alternate slats are at an offset - Enable this if the pins in alternate slats are at an offset.
